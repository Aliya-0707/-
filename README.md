# 📦 Қойма басқару жүйесі (Warehouse Management System)

## 🎯 Мақсаты:
Бұл жобаның мақсаты — коллекциялар (Map, List) арқылы деректерді басқаруды үйрену. Тауарлармен CRUD операцияларын орындау.

---

## 📌 Жобаның сипаттамасы:
Жүйеде әр тауардың:
- атауы  
- саны  
- бағасы  

сақталады. Пайдаланушы:
- Тауар қоса алады  
- Жаңарта алады  
- Өшіре алады  
- Іздей алады  

Тауарлар HashMap арқылы сақталады.

---

## ❓ Сұрақтар және жауаптар:

### 1. Map пен List айырмашылығы неде?
- List — элементтер тізімі, индекс арқылы қол жеткізіледі  
- Map — кілт-мән (key-value) құрылымы, кілт арқылы ізделеді  

---

### 2. Қай кезде Map қолдану тиімді?
Map деректерді жылдам іздеу қажет болғанда тиімді. Мысалы, тауар атауы арқылы бірден табу керек болса.

---

### 3. CRUD деген не?
CRUD — негізгі операциялар:
- Create — қосу  
- Read — оқу/көру  
- Update — жаңарту  
- Delete — өшіру  

---

## 📊 Қорытынды:
Бұл жоба коллекциялармен жұмыс істеуді жақсартады. HashMap қолдану арқылы деректерді тиімді сақтау және жылдам іздеу мүмкіндігі пайда болады. CRUD операциялары толық жүзеге асырылады.
